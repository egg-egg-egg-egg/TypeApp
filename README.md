# 介绍

这是一个C++实现的控制台打字软件, 出于教学目的设计。

整个项目会从一个基础的控制台打字软件开始，逐步完善，最终实现一个功能完善的打字软件。项目的目标是通过实现一个打字软件，学习C++的各种知识点。

并且每个版本都会逐步更新到`typeApp[n]`文件夹中。
